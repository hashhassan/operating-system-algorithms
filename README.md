# Operating System Algorithms in Java

A collection of Operating System algorithms implemented in Java using Swing for graphical user interfaces.

## Projects Included

### 1. CPU Scheduling Algorithms
Implementation of common CPU scheduling techniques:
- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Priority Scheduling
- Round Robin (RR)

Features:
- Process management
- Waiting Time calculation
- Turnaround Time calculation
- Response Time calculation

### 2. Banker's Algorithm
Implementation of the Banker's Algorithm for deadlock avoidance.

Features:
- Allocation and Maximum resource input
- Need Matrix calculation
- Data storage using SQL Server
- Swing-based GUI

### 3. Paging System
Simulation of memory management using paging.

Features:
- Frame and page calculation
- Page table generation
- Frame assignment
- Logical to Physical Address translation

## Technologies Used

- Java
- Java Swing
- JDBC
- SQL Server

## How to Run

1. Open the project in your Java IDE.
2. Configure the required libraries and database connection (for Banker's Algorithm).
3. Run the main class of the desired project.
